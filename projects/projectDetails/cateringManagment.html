Catering Managment App

Application from an hotel that tracks all the events the hotel host

The app simulates a hotel that has different events that will host, you can access the event information, the customers that the hotel has, book an event, edit, insert or delete existing records.

You can also create users and set up their roles as an admin, it also has a feature to preview an excel report using the data the application has, and if the user wants, it can be downloaded as an excel spreadsheet.
